## 🎉 诚邀加入

欢迎各路大佬加盟MS，让它更健壮的走下去。
目前MS坑位有限，技术大佬优先加入！只要技术栈是vue、python、flutter、swift、GO、会写NT规则、会基本的Html+CSS、会站点适配的其中一种的大佬，均可优先加入。
满足条件的大佬可在[Issue](https://github.com/xylplm/media-saber-wiki/issues)留下邮箱，接收邀请。
