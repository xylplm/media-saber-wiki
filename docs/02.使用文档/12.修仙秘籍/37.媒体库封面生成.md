# 🎨 媒体库封面生成

**媒体库封面生成** 插件用于为你的电影、剧集等媒体库自动生成美观的封面图片，支持多种样式和自定义配置，让你的媒体库更「好看」也更容易识别。

## ⚙️ 功能特点

- 自动扫描指定媒体库目录
- 基于名称 / 元数据生成封面
- 提供多种封面样式与颜色风格（以实际实现为准）
- 支持定时批量生成 / 更新

## 🚀 配置指南

在插件配置页面中可以看到所有相关字段，这里从使用角度给出通用说明：

### 📂 媒体路径

1. **媒体库路径**
   - 必填
   - 指向电影/剧集等媒体文件所在目录
   - 支持本地路径或挂载路径

2. **匹配规则 / 目录结构**（如表单中有相关字段）
   - 根据个人目录组织方式配置，例如：
     - `电影/片名 (年份)`
     - `剧集/剧名/Season 01` 等

### 🎭 风格与输出

3. **封面模板 / 样式**
   - 选填
   - 可选择不同风格（简约、海报感、信息密集等）

4. **输出路径 / 覆盖策略**
   - 可以选择直接写入媒体目录（如 `folder.jpg` / `poster.jpg`）
   - 或输出到独立目录

### ⏰ 调度与通知

5. **周期 (cron)**
   - 选填
   - 使用 Cron 表达式配置刷新频率，例如：
     - `0 4 * * *`：每天凌晨 4 点扫描更新

6. **消息推送 (notice)**（如表单中提供）
   - 选填
   - 批量生成完成后推送汇总统计（成功数量 / 失败条目等）

### 🔧 工具栏操作

- **运行一次 (run)**：立刻扫描媒体库并生成/更新封面。

## 📖 使用说明

1. 确认媒体库目录结构已基本稳定，避免频繁大规模变动。
2. 在插件中配置媒体库路径及必要的匹配规则。
3. 选择喜欢的封面样式与输出策略（覆盖原有封面或单独生成）。
4. 可先通过「运行一次」在少量目录上测试效果，确认样式与命名符合预期。
5. 根据媒体更新频率设置合适的 `cron`，如每日或每周批量刷新一次。

## ⚠️ 注意事项

- 若勾选覆盖模式，请确认旧封面已备份或不再需要。
- 大量媒体库首次生成可能耗时较长，建议安排在低峰时段执行。
- 如需个性化模板，可关注后续文档或插件配置说明的更新。
