# 🏢 企业微信可信 IP

**企业微信可信 IP** 插件用于定期将当前公网 IP 同步到企业微信管理后台的「可信 IP 列表」，解决没有固定公网 IP 时，频繁变更导致的接口调用失败问题。

## ⚙️ 功能特点

- 自动获取当前服务器公网 IP
- 通过企业微信开放接口更新企业可信 IP 列表
- 支持定时执行，保持 IP 白名单始终最新
- 支持消息推送，便于查看更新结果

## 🚀 配置指南

在插件配置页面中可以看到所有企业微信相关字段，一般包括：

### 📋 企业微信配置

1. **企业 ID / CorpID**
2. **应用凭证（Secret 等）**
3. **目标应用或安全设置项 ID**

> 请参照企业微信后台的实际配置项填写，确保有权限修改可信 IP。

### ⚙️ 调度与通知

1. **周期 (cron)**
   - 选填
   - 使用 Cron 表达式设置执行周期，例如：
     - `*/30 * * * *`：每 30 分钟更新一次
     - `0 * * * *`：每小时更新一次

2. **消息推送 (notice)**
   - 选填
   - 开启后，每次更新结果会通过通知渠道推送（成功 / 失败、更新的 IP 等）

### 🔧 工具栏操作

- **运行一次 (run)**：立刻执行一次 IP 获取与白名单同步操作。

## 📖 使用说明

1. 在企业微信管理后台开启并配置好可写可信 IP 的相关应用或接口权限。
2. 在插件配置中填入企业 ID、Secret 等必要信息。
3. 设置合理的 `cron` 周期，根据公网 IP 变动频率决定更新频率。
4. 建议开启「消息推送」，方便第一时间知道 IP 是否同步成功。
5. 点击「运行一次」测试是否能正确写入企业微信可信 IP 列表。

## ⚠️ 注意事项

- 频繁修改可信 IP 可能受到企业微信接口频率限制，请合理设置执行周期。
- 若服务器处于多层代理之后，请确认获取到的是对外真实出口 IP。
- 如企业微信接口或权限策略变更，请根据官方文档及时调整配置。
