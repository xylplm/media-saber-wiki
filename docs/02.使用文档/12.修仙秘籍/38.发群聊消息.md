# 📢 发群聊消息

**发群聊消息** 插件用于自动向站点的群聊（Shoutbox / Chatbox）区域发送消息，可用于定时公告、自动提示、气氛活跃等场景。

## ⚙️ 功能特点

- 自动向站点群聊区发送文本消息
- 支持定时发送
- 可配置消息内容与发送频率

## 🚀 配置指南

在插件配置页面中可以看到所有字段，这里按常见用法简单说明：

### 📋 基础配置

1. **站点选择 / 认证信息**
   - 指定要发送群聊消息的站点
   - 需要确保该站点账号已配置且有权限在群聊区发言

2. **消息内容**
   - 支持纯文本（如表单支持，也可能支持简单格式）
   - 建议避免包含容易被视为垃圾信息的内容

3. **周期 (cron)**
   - 选填
   - 使用 Cron 表达式配置发送频率
   - 示例：
     - `0 9 * * *`：每天 9 点发一次早安消息
     - `0 */3 * * *`：每 3 小时发一次提示（不推荐太频繁）

4. **消息推送 (notice)**（如表单中提供）
   - 选填
   - 开启后，在发送失败或异常时可通过通知提醒你检查配置。

### 🔧 工具栏操作

- **运行一次 (run)**：立即向指定站点的群聊区发送一次消息，用于测试配置是否正确。

## 📖 使用说明

1. 确认站点账号有群聊发言权限，且未被禁言。
2. 在插件中选择目标站点并填写消息内容。
3. 根据需求设置 `cron`，避免高频打扰其他用户。
4. 可先通过「运行一次」进行测试，确认消息能正常发送并显示。

## ⚠️ 注意事项

- 请勿将本插件用于刷屏或骚扰行为，以免导致账号被封禁或影响站点声誉。
- 如站点对群聊有防刷机制（例如限频、人机校验等），本插件可能无法绕过。
- 建议仅在合适的场景中使用（如公告、维护通知、活动提示等）。
