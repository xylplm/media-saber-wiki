# ⏱️ 时刻

**时刻** 插件用于在特定时间点提醒你那些值得被记住的瞬间，可以视作一个轻量级「纪念日 / 倒计时 / 提醒」工具。

## ⚙️ 功能特点

- 可配置多个「时刻」事件
- 支持按时间点或周期提醒
- 支持通过通知渠道发送提醒内容

## 🚀 配置指南

在插件配置页面中可以看到所有「时刻」相关字段，这里从使用角度给出通用说明：

### 📋 事件配置

1. **事件标题 / 名称**
   - 必填
   - 用于描述这个时刻，例如「纪念日」「服务器续费日」等

2. **时间 / 日期配置**
   - 必填
   - 可配置为具体日期时间，或按 cron 表达式周期触发（视实现而定）

3. **提醒内容**
   - 选填
   - 支持自定义提醒文案，例如：
     > 又是新的一年，记得看看你的媒体库啦~

### ⏰ 调度与通知

4. **周期 (cron)**（如采用 cron 方式）
   - 使用 Cron 表达式设置提醒频率
   - 示例：`0 9 * * *` 每天 9 点提醒一次

5. **消息推送 (notice)**
   - 选填
   - 开启后会通过通知渠道发送提醒内容，适合与手机 / IM 集成

### 🔧 工具栏操作

- **运行一次 (run)**：立即触发一次提醒（若实现支持），用于测试通知渠道是否配置正确。

## 📖 使用场景示例

- 重要纪念日提醒（生日、纪念日等）
- 服务器 / 域名 / 会员到期提醒
- 定期回顾 / 总结提示（每周/每月）

## ⚠️ 注意事项

- 建议搭配通知渠道使用，否则只能在系统内看到提醒日志。
- 如设置了非常密集的 cron 周期，可能导致频繁提醒，注意控制频率。
